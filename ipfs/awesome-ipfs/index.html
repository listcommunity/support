Awesome IPFS

Useful resources for using IPFS and building things on top of it

This list is for projects, tools, or pretty much any things related to IPFS that are totally awesome. This is for products which are already awesome - if you have plans for cool stuff to do with IPFS, you should build it, and then link it here. If you have an idea for an awesome thing to do with IPFS, a good place to ask about it might be in ipfs/apps40 7 or ipfs/notes119 9 .

Apps

  • akasha - A Next-Generation Social Media Network, powered by Ethereum and embedded into IPFS.
  • Alexandria - Decentralized content publishing / monetization platform.
  • Arbore25 4 - A friend-to-friend file-sharing app build on top of IPFS.
  • beets3k 589 - Beets has a plugin which allows for easy sharing of music libraries using IPFS
  • Blokaly - A badge issuing, sharing and display platform based on IPFS.
  • Boards - Distributed social platform that runs in the browser. GitHub140 15
  • Cohort15 3 - A golang app to preset a threejs interface and get all of its assets out of IPFS.
  • Computes - Computes.io is a distributed supercomputer powered by IPFS.
  • dapple155 47 - Dapple is a Solidity developer multitool designed to manage the growing complexity of interconnected smart contract systems.
  • digx - Digix is an asset-tokenisation platform built on Ethereum and IPFS.
  • dtube - Distributed video sharing with steem.it intergrations, using ipfs for backend storage.
  • Ethlance - First completely decentralised job market platform built on Ethereum and IPFS. Github60 13
  • git-ipfs-rehost - A script to rehost your git repos in ipfs.
  • Global Upload - File transportation service for IPFS, upload files to the future of distributed web.
  • HydrusNetwork501 71 - A booru-style media tagging application with a multitude of features, recently added basic ipfs support.
  • InterPlanetary Wayback74 10 - Web Archive (WARC) indexing and replay using IPFS.
  • Interplanetary Wiki44 2 - Wiki built on top of IPFS
  • IPFS Event Drop5 1 - An app that allows for improved visualization of ipfs events.
  • ipfs.ink - Publish and render markdown essays to and from ipfs. GitHub
  • ipfs-search - Search engine for files and directories on IPFS.
  • ipfs-share - Pastebin/Image host/File sharing application
  • ipfs.pics - Upload and share pics.
  • IPFSBin - Pastebin clone build.
  • IPFSStore - Pinning paid with Bitcoin
  • markup.rocks - Pandoc-based markup editor/previewer/converter, ported to IPFS. Example
  • Orbit767 54 - Distributed, peer-to-peer chat application on IPFS.
  • Partyshare - A simple file sharing app.
  • Playback - IPFS playback support. This allows casting a video in IPFS to a Chromecast.
  • PushToTalk - Push to Talk lets you edit audio essays and publish them with IPFS.
  • ujo - A blockchain marketplace for musicians.
  • uport - Uport is a mobile, self-sovereign identity and key management system, built on the Ethereum blockchain.
  • gogo.tattoo - Gogo Tattoo Project uses IPFS, DLTs and other modern technologies to offer tattoo artists and wearers an unbreakable lifetime record of their portfolios. gogo.tattoo app already has a feature to share works directly to IPFS.

Tools

  • cachewarmer - Donate ipfs gateways to cache other people's content
  • git-remote-ipfs96 4 - push/pull repositories from/to IPFS
  • http2ipfs20 1 - This is a simple webtool to add URLs to an IPFS node.
  • ipcat - Retrieve IPFS object data and send it to stdout.
  • ipfs-chrome-station - Chrome extension to redirect ipfs.io traffic to local gateway
  • ipfs-chrome-extension59 5 - Chrome extension to redirect ipfs.io traffic to local gateway
  • ipfs-companion29 10 - Firefox addon to provide access to IPFS via local gateway
  • ipfs-gui18 2 - Windows UI integration and IPFS installer
  • ipfs-linux-service33 6 - IPFS Linux Init Daemon
  • ipfs-paste18 2 - Paste stdin and clipboard to IPFS
  • ipfs-screencap27 3 - Capture screenshots, publish them to IPFS, and copy the link to the clipboard.
  • ipfscrape - Scrape a webpage with all assets and put it in IPFS
  • ipget36 7 - wget for IPFS: retrieve files over IPFS and save them locally.
  • IPRedirect - Browser userscript for redirecting IPFS/IPNS addresses to your local gateway. This should work on any browser that hasnt had an extension written for it yet and has support for userscripts.
  • ipscend139 12 - Tool for hosting web apps and static websites in IPFS
  • pinbot18 10 - Pin content via IRC
  • ipfs-mount8 1 - Mount IPFS as a mapped drive on Windows
  • ipfs-add-from-url - Add a file to IPFS from a URL instead of a file path
